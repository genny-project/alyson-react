# Alyson v2
## Installation and up and running for `IOS`
1. `./git-pull-all-projects.sh`
2. `./run-ios.sh`
## Installation and up and running for `Web`

1. `./git-pull-all-projects.sh` 
2. `./run-web.sh`

## Manual Installation
1. `git pull `
2. `git checkout js-convert` 
3. `npm install`
4. `react-native run-ios`
5. for web do  `npm run start-web`
