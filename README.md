# Alyson v2

## Installation

1. Download

```git clone https://github.com/genny-project/alyson-v2.git alyson-v2```

2. Navigate to root folder

```cd alyson-v2```

3. Install dependancies

```npm install```

4. Run

```npm run start-web```

5. Run on iOS 
 ``` npm run start-ios ``` 


### Notes 

1. Entry point for native project is ```index.js``` on root folder
